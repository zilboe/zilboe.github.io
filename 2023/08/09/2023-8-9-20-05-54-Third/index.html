嵌入式c语言【第一讲】-基本数据类型

开篇

嵌入式里常用c语言，如果是linux开发，也会常用c++和其他的一些语言，不过为了基础一点，从c语言开始记录。

嵌入式c语言开篇，常用作通过编程来控制硬件，当前编程语言很多，与硬件打交道的基本都是c语言，51，stm32的一些硬件，可以编写程序烧录到芯片内部的flash(存储区)来让硬件自己执行程序，达到你想要的效果。

当然当前编程语言太多了，不过我还是打算从c基础开篇开始记录下。

前提环境

我的建议是有虚拟机vmware或者是在linux系统下编程，如果不知道linux，可以从网上找vmware软件来下载，然后从ubuntu官网下载一个系统镜像安装到vmware中，具体操作步骤可以自行从网上搜索。当然如果你有云服务器也可以，只要是linux操作系统就行，因为涉及到编译等环境搭建很简单。

c语言基本类型

在基本类型之前我要说明几点：

关键字，编程语言都有所谓的关键字，关键字就是告诉机器你句话是什么意思，他好按照你给他的关键字来将你写好的程序编译成机器语言好去执行，机器语言就是只有机器能读懂的语言，也就是二进制格式，0和1，机器底层只能执行0和1，所以要按照你给的关键字将代码最终整理成0和1。

c语言的关键字有：

$\qquad$ auto ：声明自动变量

$\qquad$ break：跳出当前循环

$\qquad$ case：开关语句分支

$\qquad$ char：声明字符型变量或函数返回值类型

$\qquad$ const ：声明只读变量

$\qquad$ continue：结束当前循环，开始下一轮循环

$\qquad$ default：开关语句中的“默认”分支

$\qquad$ do ：循环语句的循环体

$\qquad$ double ：声明双精度浮点型变量或函数返回值类型

$\qquad$ else ：条件语句否定分支（与 if 连用）

$\qquad$ enum ：声明枚举类型

$\qquad$ extern：声明变量或函数是在其它文件或本文件的其他位置定义

$\qquad$ float：声明浮点型变量或函数返回值类型

$\qquad$ for：一种循环语句

$\qquad$ goto：无条件跳转语句

$\qquad$ if：条件语句

$\qquad$ int： 声明整型变量或函数

$\qquad$ long ：声明长整型变量或函数返回值类型

$\qquad$ register：声明寄存器变量

$\qquad$ return ：子程序返回语句（可以带参数，也可不带参数）

$\qquad$ short ：声明短整型变量或函数

$\qquad$ signed：声明有符号类型变量或函数

$\qquad$ sizeof：计算数据类型或变量长度（即所占字节数）

$\qquad$ static ：声明静态变量

$\qquad$ struct：声明结构体类型

$\qquad$ switch :用于开关语句

$\qquad$ typedef：用以给数据类型取别名

$\qquad$ unsigned：声明无符号类型变量或函数

$\qquad$ union：声明共用体类型

$\qquad$ void ：声明函数无返回值或无参数，声明无类型指针

$\qquad$ volatile：说明变量在程序执行中可被隐含地改变

$\qquad$ while ：循环语句的循环条件

乍一看很多，其实常用的也就几个，循环语句使用while，for

判断使用：if，或者三目运算符，三目运算符到后边再讲。

还有条件多的情况下使用：switch case，比if速度要快。

还有基本的类型声明

$\qquad$ 1.整型：int

$\qquad$ 2.字符型：char

$\qquad$ 3.短整型：short

$\qquad$ 4.长整型：long

$\qquad$ 5.长长整型：long long

$\qquad$ 6.浮点型：float

$\qquad$ 7.双精度浮点型：double

以上声明的数据类型都是有符号的，无符号的在前边声明unsigned。

例子：

$\qquad$ 声明一个整型变量a：int a;

$\qquad$ 一个字符类型变量b：char b;

$\qquad$ 一个短整型变量c：short c;

等等，无符号类型就是在前边加上unsigned来修饰

$\qquad$ 一个无符号整型变量a：unsigned int a;

$\qquad$ 一个无符号字符类型变量b：unsigned char b;

$\qquad$ 一个无符号短整型变量c：unsigned short c;

有符号与无符号

有符号的意思是这个变量的空间的最高位，代表着该数是正数还是负数

一个int类型的变量，它在程序空间里边占着四个字节的单位。

我们的机器，最底层内部结构其实是分布很多的0与1的小空间。这个小空间只有高与低，就像一个只能装一滴水的杯子，要么是空杯子0，要么就是装水的杯子(一滴水)1。这些空间(杯子)八个连在一起。
